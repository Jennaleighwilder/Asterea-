How It Works

Court-Safe Workflow

Observation to export in four steps

1. Create Session

Open a case session with automatic expiration. Data is ephemeral by design.

2. Log Observations

Record panel dynamics, question responses, and body signals using neutral, observable language only. System blocks interpretive or predictive terms.
